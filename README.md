# Leitor de Contatos e Disparo WhatsApp

Este repositório contém a solução desenvolvida para o desafio de Estágio em Desenvolvimento Python da **b2bflow**. O objetivo é integrar um banco de dados Supabase com a Z-API para automatizar o envio de mensagens de WhatsApp.

## 🛠️ Tecnologias Utilizadas
- **Python 3.10+**
- **Supabase** (Banco de Dados PostgreSQL)
- **Z-API** (Gateway de mensageria WhatsApp)
- **python-dotenv** (Gerenciamento de variáveis de ambiente)

## 🚀 Como Executar

### Pré-requisitos
- Python instalado na sua máquina.
- Conta ativa no [Supabase](https://supabase.com/) e na [Z-API](https://z-api.io/).
